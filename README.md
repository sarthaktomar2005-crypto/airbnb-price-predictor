# 🏠 Airbnb Price Prediction App

## 📌 Project Overview
This project predicts Airbnb listing prices based on various features like number of reviews, availability, rating, and more.

It uses Machine Learning and is deployed using Streamlit.

---

## 🚀 Features
- Predict Airbnb prices instantly
- Clean and user-friendly UI
- Built using real-world dataset
- Includes Power BI Dashboard for insights

---

## 🧠 Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit
- Power BI
